The Canvas

About

Welcome to JavaScript Turtle! Turtle is a class "learn to program" tool, in which a simple shape (the "turtle") is used to draw on a target screen. In the language of choice, simple commands like "forward" or "right" are entered to make the turtle move.

From simple building blocks, more complicated shapes can be built. Can you build a staircase? A square? What about a general function for a polygon of any shape?

Command Reference

Movement
forward(distance)Move the "turtle" forward by "distance" (in pixels).
right(angle)Rotate the turtle clockise by "angle" (in degrees).
left(angle)Rotate the turtle counter-clockwise by "angle" (in degrees).
goto(x,y)Position the turtle at the specified x,y coordinates. Note that 0,0 is the center of the canvas.
Canvas and drawing
clear()Erase all drawing on the canvas. The turtle stays in place.
penup()Configure the turtle to move without drawing.
pendown()Configure the turtle to draw as it moves.
reset()Reset the canvas and turtle to a pristine state.
angle(angle)Set the drawing direction; this does not rotate the turtle.
width(width)Set the width of the line drawn by the turtle as it moves.
colour(r,g,b,a)Set the color of the line drawn by the pen.
write(string)Write "string" at the current turtle location.
n = random(low,high)Generate a random number (and assign it to the variable n).
hideTurtle()Make the turtle disappear.
showTurtle()Make the turtle re-appear.
redrawOnMove(bool)
draw()
repeat(n, action)Repeat "action", "n" times.
wrap(bool)Turn edge wrapping on or off.
Animation
animate(action,ms)Execute the function "action" every "ms" milliseconds.
stopAnimation()Cause any previous animation to stop.
